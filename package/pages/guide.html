Hawk Pass User Guide

Welcome to Hawk Pass! This guide will help you use all features of the password generator extension.

Getting Started

  1. Install the Hawk Pass extension from the Chrome Web Store or GitHub.
  2. Click the Hawk Pass icon in your Chrome toolbar to open the popup.
  3. Use the popup to generate passwords or adjust settings.
  4. On any website, Hawk Pass will add a Generate button next to password fields for quick password insertion.

Features

  • Single Password Generation: Click Generate Password to create a secure password. It will be copied to your clipboard automatically.
  • Password Strength Indicator: Shows the strength of your generated password based on length and character types.
  • Bulk Password Generation: Enter the number of passwords you need and click Generate Lists. A CSV file with all passwords will be downloaded.
  • Website Integration: On any site, a Generate button appears next to password fields. Click it to auto-fill a strong password.

Password Settings

Customize how your passwords are generated in the Settings menu:

  • Password Length:
    • Set from 6 to 64 characters using the slider.
    • Recommended: at least 12 characters for strong security.
  • Character Types:
    • Big letters (A-Z): Include uppercase letters.
    • Small letters (a-z): Include lowercase letters.
    • Numbers (0-9): Include digits.
    • Special chars (!@#$...): Include special characters.
  • Saving Settings: After changing any option, click Save to apply your preferences.

Password Strength Indicator

  • Very weak: Only one character type or short length
  • Weak: Two character types
  • Good: Three character types
  • Strong: Four character types
  • Very strong: Four types and length at least 10

Website Integration

  • When you visit a website with a password field, Hawk Pass automatically adds a Generate button next to it.
  • Click the button to instantly generate and fill a strong password.
  • The button uses a unique style for visibility and consistency.

Tips for Secure Passwords

  • Use at least 12 characters for important accounts.
  • Enable all character types for maximum strength.
  • Never reuse passwords across different sites.
  • Store your passwords in a secure password manager.

Troubleshooting

  • If the extension does not work on a website, try refreshing the page.
  • Make sure you have granted the extension permissions in Chrome.
  • For more help, visit the GitHub repository.
